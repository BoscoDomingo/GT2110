USERS:
Lista de usuarios. Cada usuario acaba en un slash ( / ). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , ).
	0 - ID (a0000)
	1 - Email
	2 - Password
	3 - Nombre
	4 - Apellido
	5 - Alias
	6 - Lista de IDs de usuarios seguidos
	7 - lista de IDs de seguidores
	8 - Lista de IDs de Comunidades
	9 - Lista de IDs de Publicaciones
	10 - 

TWEETS:
Lista de tweets. Cada tweet acaba en un slash ( / ). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , )
	0 - ID (b0000)
	1 - Tipo (T, E ó R)
	2 - ID Usuario que ha posteado
	3 - Fecha
	4 - Hora
	5 - Numero Likes
	6 - Num dislikes
	7 - Lista de IDs de Valoraciones
	8 - Texto
	9 - ID tweet al que referencia (Si es referencia) o Enlace externo (si es de enlace). Null si es texto
