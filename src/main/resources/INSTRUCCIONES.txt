USERS:
Lista de usuarios. Cada usuario acaba en retorno de línea(ENTER). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , ).
	0 - ID (a0000)
	1 - Email
	2 - Password
	3 - Nombre
	4 - Apellido
	5 - Alias
	6 - Lista de IDs de usuarios seguidos
	7 - lista de IDs de seguidores
	8 - Lista de IDs de Comunidades
	9 - Lista de IDs de Publicaciones ;
	10- Lista de IDs de Valoraciones

PUBLICACIONES:
Lista de publicaciones. Cada tweet acaba en retorno de línea(ENTER). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , )
	0 - ID (b0000)
	1 - Tipo (T, E ó R)
	2 - ID Usuario que ha posteado
	3 - Fecha y Hora (formato dd/MM/yyyy HH:mm::ss)
	4 - Texto
	5 - Numero Likes
	6 - Numero Dislikes
	7 - Lista de IDs de Valoraciones
	8 - Lista de IDs de Comentarios
	9 - ID tweet al que referencia (si es Referencia) o Enlace externo (si es de Enlace). 0 si es Texto

COMENTARIOS:
Lista de comentarios. Cada comentario acaba en retorno de línea(ENTER). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , )
	0 - ID (c0000)
	1 - Fecha y Hora (formato dd/MM/yyyy HH:mm::ss)
	2 - Texto
	3 - ID Publicacion a la que pertenece
	4 - ID Usuario que ha posteado

VALORACIONES:
Lista de valoraciones. Cada valoración acaba en retorno de línea(ENTER). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , )
	0 - ID (d0000)
	1 - Tipo (1 positiva, 0 negativa)
	2 - ID Usuario que ha posteado
	3 - ID Publicacion a la que pertenece

COMUNIDADES:
Lista de comunidades. Cada comunidad acaba en retorno de línea(ENTER). Todos sus campos irán separados por punto y coma ( ; ) y cada valor de un campo, en caso de tener múltiples, irá separado por comas ( , )
	0 - ID (e0000)
	1 - Nombre
	2 - Lista de IDs de miembros
	3 - Lista de IDs de admins
